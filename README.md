# Patrick David Portfolio Website

A recreation of the minimal and brutalist portfolio website for Patrick David, a UX/UI designer and developer based in Italy.

## Features

- **Minimal & Brutalist Design**: Clean, bold typography with black and white aesthetic
- **Responsive Layout**: Fully responsive design that works on all devices
- **Smooth Animations**: Scroll-triggered animations and interactive elements
- **Modern Interactions**: Hover effects, smooth scrolling, and parallax effects

## Project Structure

```
.
├── index.html      # Main HTML structure
├── styles.css      # All styling and responsive design
├── script.js       # JavaScript for interactions and animations
└── README.md       # This file
```

## Getting Started

1. Simply open `index.html` in your web browser
2. No build process or dependencies required - it's pure HTML, CSS, and JavaScript

## Sections

- **Navigation**: Fixed navigation with links to works, about, and contact
- **Hero**: Large typography introduction with contact buttons
- **About**: Personal introduction and background
- **Works**: Selected project cases
- **Project Details**: Detailed information about featured projects
- **Awards**: Showcase of awards and recognition
- **Contact**: Contact section with interest tags

## Design Notes

- Uses Inter font family for clean, modern typography
- Black and white color scheme
- Large, bold typography for headings
- Minimal use of color and effects
- Brutalist design principles with strong borders and clear hierarchy

## Browser Support

Works on all modern browsers (Chrome, Firefox, Safari, Edge).

## Customization

You can easily customize:
- Colors in the `:root` CSS variables
- Typography by changing the font-family
- Content by editing the HTML
- Animations in the JavaScript file
